# App Description — japanese-study

## 1. 앱 개요

| 항목 | 내용 |
|------|------|
| **앱 이름** | japanese-study |
| **플랫폼** | iOS (SwiftUI) |
| **목적** | 일본어 초보자가 문자(히라가나·가타카나)부터 체계적으로 학습 |
| **언어** | UI: 한국어 · 학습 콘텐츠: 일본어(히라가나/가타카나) + 로마자 |

한국어 사용자가 일본어를 처음 배울 때 가장 먼저 막히는 **문자 읽기**를 해결하는 것이 1차 목표입니다.  
단계별 챕터 구조로 기본 음절 → 특수음 → 가타카나 순으로 확장합니다.

---

## 2. 컨셉

### 한 줄 정의

> **차분하고 명확한 UI로, 매일 조금씩 일본어 문자를 익히는 학습 앱**

### 핵심 가치

1. **단계적 학습** — 챕터 단위로 범위를 나누어 부담 없이 진행
2. **시각적 명료함** — 레퍼런스(`layout_01`)처럼 여백·카드·색 대비를 활용해 정보 밀도 조절
3. **일본어 맥락 유지** — 메뉴·제목에 ひらがな / カタカナ 표기를 병기해 실제 문자에 익숙해지도록 유도
4. **반복 학습 친화** — 추후 플래시카드·퀴즈·발음 재생 등으로 복습 루프 구성

### 타깃 사용자

- 일본어를 처음 시작하는 한국어 사용자
- 히라가나는 알지만 장음·촉음·요음이 헷갈리는 사용자
- 짧은 시간(5~10분) 단위로 문자만 따로 공부하고 싶은 사용자

## 4. 챕터 상세 (기획)

### Ch.01 — 기본 히라가나

- **범위**: あ행 ~ わ행, ん (기본 46자)
- **학습 요소**
  - 50음도 표 (행·단 구분)
  - 문자 · 로마자 · 발음 안내
  - 탭 시 발음 재생 (TTS, 추후)
- **UI 형태**: 레퍼런스 카드 그리드 — 문자 카드 + 하단 로마자

### Ch.02 — 특수음

- **범위**
  - **장음 (長音)**: 모음 연장 (例: おかあさん)
  - **촉음 (促音)**: っ 로 음절 앞 쉼 (例: がっこう)
  - **요음 (拗音)**: ゃ·ゅ·ょ 결합 (例: きょう)
- **학습 요소**
  - 규칙 설명 (한국어)
  - 예시 단어 카드
  - 읽기 연습 / 미니 퀴즈

### 가타카나 (예정)

- 히라가나와 동일한 챕터 번호·구조
- Ch.01 기본 가타카나, Ch.02 특수음
- 외래어·강조 표기 맥락 소개
